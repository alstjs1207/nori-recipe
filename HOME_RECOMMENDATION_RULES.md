# Home Recommendation Rules

홈 추천 관련 작업은 이 문서를 기준으로 판단한다. 아래 규칙은 임의로 바꾸지 않는다.

## 고정 규칙

1. 홈 추천 놀이는 재료 기준으로 항상 3개만 구성한다.
2. 3개 중 재료 정합도가 가장 높은 1개를 상단 `오늘의 추천 놀이`로 노출한다.
3. 한번 정해진 추천 3개는 3개 모두 완료되기 전까지 홈 화면에서 바뀌지 않는다.
4. 완료된 놀이는 목록 안에서 완료 상태가 명확하게 보여야 한다.
5. 추천 3개를 모두 완료한 뒤 홈 화면에 진입하면 `오늘의 놀이를 모두 완료했어요` 메시지와 함께 폭죽 애니메이션을 보여준다.
6. 사용자가 재료를 편집하면 기존 추천 세트는 폐기하고 새로운 추천 3개를 다시 만든다.

## 작업 원칙

- 홈 추천 관련 UI, 상태, 추천 로직을 수정할 때마다 먼저 이 문서를 확인한다.
- 위 규칙과 충돌하는 변경은 그대로 진행하지 않고, 먼저 규칙 변경 여부를 확인한다.
- 홈 화면 구현 리뷰 시에도 이 문서를 기준으로 동작이 유지되는지 확인한다.
